Question No: 3 What is the different of local storage and session storage?
Local storage Session storage
Size of local storage can be up to 10mb. Some say its size is 5mb. Size Storage size is up to 5mb.
If you do not delete from the browser, it can not be deleted automatically. Closing the browser or closing the tab clears the storage.
Local storage does not work for any specific tab. It works for any tab. Session Storage works for the specified tab and when the tab is closed the data is deleted.
